# InfiniteKeyboard — 2オクターブ無限音階キーボード

PCのキーボードで演奏できる「無限音階（シェパードトーン）」キーボードです。
音域の異なるサイン波を重ね合わせることで、上がり続けても／下がり続けても
キリがないように聞こえる音響錯覚を、鍵盤として弾けるようにしています。

**デモ:** https://masui.github.io/InfiniteKeyboard/

## 使い方

1. ページを開いて、最初に画面を一度クリックします（ブラウザの仕様で、音を出すにはユーザー操作が必要です）。
2. キーボードのキーを押すと音が鳴ります。

| 段 | キー | 役割 |
| --- | --- | --- |
| 上段（白鍵） | `Q W E R T Y U I O P [ ]` | 低めのオクターブの音階 |
| 数字段（黒鍵） | `2 4 5 7 8 9` | 上段の半音（シャープ） |
| 下段 | `Z S X C F V G B N J M K , L . /` | 高めのオクターブの音階 |

上段と下段は音域が重なっており、たとえば上段の `E` で出るド音と下段の `/` で出るド音は
同じ高さに聞こえます。これが「無限音階」が滑らかにループする仕掛けです。

> キー配列は MacBook（US配列）のキーボードを想定しています。
> `[ ] - =` などの記号キーは配列によって入力される文字が異なるため、
> 他のキーボードでは一部のキーが反応しないことがあります。

## 仕組み

1つの音について、基準周波数を中心に複数オクターブ分のサイン波を同時に鳴らし、
440Hz から離れるほど音量が小さくなる釣鐘状のゲインをかけています
（[シェパードトーン](https://ja.wikipedia.org/wiki/無限音階)）。
これにより、隣の音へ移動しても端の音が滑らかにフェードイン／アウトし、
音階が途切れずにつながって聞こえます。

実装は [Web Audio API](https://developer.mozilla.org/ja/docs/Web/API/Web_Audio_API)
（`OscillatorNode` / `GainNode`）を素のJavaScriptで使っているだけで、外部ライブラリは不要です。

## ファイル構成

| ファイル | 内容 |
| --- | --- |
| `index.html` | 画面とスタイル。`keyboard.js` を読み込むだけ |
| `keyboard.js` | キー配列・音名・周波数の定義と発音処理 |

## 開発・動作確認

ビルドは不要です。リポジトリを clone して `index.html` をブラウザで開くか、
簡易サーバーで配信してください。

```sh
python3 -m http.server
# → http://localhost:8000/ を開く
```
